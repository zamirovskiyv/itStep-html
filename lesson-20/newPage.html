Справочник по HTML-тегам

<h1> — Заголовок первого уровня

Используется для самого важного заголовка страницы.

<h1>Это заголовок</h1>

<p> — Абзац

Используется для создания текста абзацами.

<p>Это абзац текста.</p>

<mark> — Выделенный текст

Помечает текст, который нужно выделить цветом (обычно жёлтым).

<p>Это <mark>важный текст</mark>.</p>

<s> — Зачёркнутый текст

Используется для пометки устаревшей или некорректной информации.

<p>Цена: <s>5000 ₽</s> 4000 ₽</p>

<a> — Ссылка

Создаёт гиперссылку на другую страницу или сайт.

<a href=\"https://example.com\">Перейти на сайт</a>

<img> — Изображение

Добавляет картинку на страницу. Атрибут src указывает путь к файлу.

<img src=\"images/photo.jpg\" alt=\"Описание изображения\" width=\"300\">

<ul>, <li> — Список с маркерами

Создаёт маркированный список.

<ul> <li>Пункт 1</li> <li>Пункт 2</li> </ul>

<table>, <tr>, <td> — Таблица

Создаёт таблицу с ячейками и строкам
